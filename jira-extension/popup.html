Patching CORS for

*.jira.com/rest/*
Setup (one-time)
  1. Open PM Tracker and click the Jira config button (⚙)
  2. Enter your Atlassian Base URL, e.g.
    https://company.atlassian.net
  3. Enter your Atlassian email and an API token
  4. Leave Proxy URL empty — this extension handles CORS directly
Only modifies responses from Atlassian REST endpoints.
Credentials never leave your browser.
